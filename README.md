# arnis

Берёт реальные координаты с OpenStreetMap и генерирует из них Minecraft-мир.

Дороги → грунтовые пути. Здания → дома из кирпича. Парки → трава и деревья.
Вода → вода. Всё как есть, только размером 1:1 в блоках.

## Запуск

```bash
cargo run --release -- --lat 55.7558 --lon 37.6173 --radius 500
```

Параметры:
```
--lat     широта центра (например 55.7558 для Москвы)
--lon     долгота центра (например 37.6173)
--radius  радиус в метрах (по умолчанию 300)
--output  папка для .mcworld файла (по умолчанию ./world)
--scale   масштаб: сколько метров в одном блоке (по умолчанию 1)
```

## Как работает

1. Запрашивает данные у Overpass API (зеркало OSM)
2. Парсит GeoJSON: дороги, здания, воду, растительность
3. Проецирует координаты в сетку блоков
4. Генерирует структуру Minecraft Bedrock Edition
5. Сохраняет как `.mcworld` — открывается двойным кликом

## Пример

```bash
# Красная площадь
cargo run --release -- --lat 55.7539 --lon 37.6208 --radius 400

# Центр Берлина
cargo run --release -- --lat 52.5200 --lon 13.4050 --radius 600
```

## Зависимости

- Rust 1.75+
- Доступ к интернету (Overpass API)
